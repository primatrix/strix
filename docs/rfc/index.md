# RFC（技术决策记录）

RFC (Request for Comments) 用于记录重要的技术决策和架构变更提案。

## 如何提交 RFC

1. 复制 [RFC 模板](./template.md)
2. 创建新文件 `docs/rfc/NNNN-your-title.md`（NNNN 为递增编号）
3. 填写模板内容
4. 提交 PR，指定架构组为 Reviewer
